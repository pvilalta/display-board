# display-board

In order to run the project, follow the instructions below

## installation
```
npm run install:all
```
```
npm run populatedb
```

## BACKEND .env exemple

```
API_PORT="3000"
```

## FRONTEND .env exemple

```
VITE_APP_PORT=8081
VITE_API_URI="http://localhost:3000"
```
Following this exemple the frontend will be available at http://localhost:8081

## run
```
npm run dev:all
```
